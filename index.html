☁️ Синхронизация между устройствами (GitHub Gist)

Подключите GitHub Gist — и данные будут автоматически синхронизироваться между всеми вашими устройствами. Как настроить (один раз) →

🔐 Шаг 1. Создайте Personal Access Token

  1. Откройте github.com/settings/tokens (нужен аккаунт GitHub)
  2. Нажмите «Generate new token (classic)»
  3. Note: 1cdo-tracker
  4. Expiration: No expiration (или подальше)
  5. Scopes: отметьте только ✅ gist
  6. Нажмите «Generate token» и скопируйте токен (начинается с ghp_ или github_pat_) — он показывается только один раз!

📝 Шаг 2. Создайте пустой Gist

  1. Откройте gist.github.com
  2. Filename: 1cdo-tracker.json
  3. Содержимое: {} (просто две фигурные скобки)
  4. Нажмите «Create secret gist» (важно: secret, не public!)
  5. Из URL скопируйте ID Gist: gist.github.com/USER/ЭТО_ID

🔌 Шаг 3. Подключите ниже

Вставьте токен и ID Gist в поля. Эти данные хранятся локально на устройстве, никуда не отправляются (кроме самого GitHub).

• При включении: изменения отправляются в облако через 1 сек
• Облако проверяется каждые 10 сек на свежие данные с других устройств
• При возврате на вкладку (с другого приложения) — проверка облака сразу
• Подгрузка из облака происходит автоматически при открытии трекера
